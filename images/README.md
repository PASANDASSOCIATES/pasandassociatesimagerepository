# Images folder

This folder contains one subfolder for each property. Upload image files into the appropriate subfolder.

How to upload (GitHub web UI):
- Open the subfolder you want (for example images/mjejane-bush-lodge/)
- Click Add file → Upload files and drag your images into the area
- Commit the upload

Notes:
- GitHub blocks files larger than 100 MB. If you need to store very large images or many binaries, ask me to set up Git LFS for you.
